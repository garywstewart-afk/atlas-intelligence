**1. Purpose and Scope**

This brief catalogs the primary-source knowledge repositories supporting
Atlas Strategic Consultants\' research programme across academic,
geoscience, coding/ML, dataset, and foreign-language domains. It is
designed as an additive, session-persistent reference --- updated with
each new source identified --- rather than a static snapshot. All
sources listed are direct-access primary repositories. Aggregators are
noted only where no superior primary source exists.

**5. Usage Protocol**

The following access discipline applies across all repository usage
within the Atlas research programme.

-   Always query primary repositories before aggregators. Aggregators
    (Google Scholar, ResearchGate) are useful for discovery but never
    for citation --- trace back to the primary journal or institutional
    source.

-   For Australian mining tenements, start with NSW MinView or the
    relevant state portal before querying GeoRef or GSQ. State portals
    contain tenement-specific regulatory and geological data not indexed
    elsewhere.

-   For ML/AI research, arXiv cs.LG and Papers With Code are the
    canonical sources for state-of-the-art benchmarks. Hugging Face is
    the primary source for model implementations.

-   Foreign-language sources require verification of institutional vs.
    open access status before citation. HAL (French), CiNii (Japanese),
    and CyberLeninka (Russian) are fully open. CNKI and Mandumah are
    subscription-gated.

-   Loop 3D (Geoscience Australia / OneGeology) is the recommended
    open-source platform for 3D probabilistic geological modelling
    relevant to borehole interpretation.

**6. Update Protocol**

This brief is version-controlled. Each Atlas research session that
identifies a new primary source appends an entry to the relevant table
and increments the version number. Source notes are mandatory for every
entry. Obsolete or superseded URLs are struck through rather than
deleted to preserve research provenance.
